MemoryPieces Support

Need help with MemoryPieces? This page covers creating puzzles, starter puzzles, saved progress, purchases, and how to contact support.

Common topics

Creating a puzzle

Start with a favorite photo from your device or choose an included starter puzzle, then adjust the setup before solving.

Using starter puzzles

Starter puzzles let you try MemoryPieces right away with cozy included images.

Saving progress

Your puzzle progress can be saved so you can pause and continue later.

Restoring Ad-Free purchase

Use the restore purchase option in the app. Make sure you are signed in with the same Apple ID used for the original purchase.

Contact support

Email support@coldharborstudio.com for help.
